SVG to Icon Font Generator — a free, browser-based tool that converts your SVG icons into a web icon font (TTF, WOFF, WOFF2) with CSS and a demo page. Everything runs locally in your browser; nothing is uploaded. A privacy-friendly alternative to IcoMoon. You can also import an existing icon font to update it, and auto-center, trim and pad icons.

Is it free?

Yes, it is completely free and runs entirely in your browser.

Are my files uploaded to a server?

No. All processing happens locally in your browser, so your files never leave your computer.

What formats does it generate?

TrueType (TTF), WOFF and WOFF2, plus a CSS file and an HTML demo page.

Can I import and update an existing icon font?

Yes — import a TTF, OTF, WOFF or WOFF2 file; its glyphs load as editable icons and keep their original codepoints.

Can I fix an off-center icon?

Yes — enable “Center & trim” to recenter each icon by its real artwork, and use the padding slider.
