SVG to Icon Font Generator

This free online tool converts your SVG icons into a web icon font — TTF, WOFF and WOFF2 — together with a CSS file and an HTML demo cheatsheet. It is a browser-based alternative to tools like IcoMoon: everything runs locally in your browser, so your icons are never uploaded anywhere.

What you can do

  • Drag and drop SVGs and download a complete icon font (TTF + WOFF + WOFF2 + CSS).
  • Import an existing icon font (TTF/OTF/WOFF/WOFF2) and update it while keeping its codepoints.
  • Automatically center, trim and pad icons so off-center artwork sits correctly.
  • Work offline and privately — no sign-up, no server, no file uploads.

Frequently asked questions

Is it free?

Yes, it is completely free and runs entirely in your browser.

Are my files uploaded to a server?

No. All processing happens locally in your browser, so your files never leave your computer.

What formats does it generate?

TrueType (TTF), WOFF and WOFF2, plus a CSS file and an HTML demo page.

Can I import and update an existing icon font?

Yes — import a TTF, OTF, WOFF or WOFF2 file; its glyphs load as editable icons and keep their original codepoints.

Can I fix an off-center icon?

Yes — enable “Center & trim” to recenter each icon by its real artwork, and use the padding slider.
